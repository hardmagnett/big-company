Чтобы было меньше зависимостей, решил скачать шрифт и CSS для него напрямую.
Файл .css переименован в .scss, чтобы легко подключился к сборщику.
В .css-файле исправлен путь к шрифту.

Шрифт лучше чем svg-иконки. Его можно:
- напрямую вставлять в текст
- легко раскрасить в нужный цвет, работая как с текстом
- легко сделать нужного размера, работая как с текстом
