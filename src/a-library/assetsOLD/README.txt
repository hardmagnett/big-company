Часть переменных CSS, часть переменных SCSS.
Потому-что у них разные возможности.
                                      CSS         SCSS
Можно использовать из JS              +           -

Можно передать в качестве параметра   -           +
в SCSS-миксин или SCSS-функцию

Можно использовать в MediaQuery       -           +

Можно задать разное значение
в зависимости от MediaQuery           +           -

Некоторые CSS-переменные создаются на основе SCSS-переменных,
чтобы избежать дублирования обьявления одинаковых значений.
Например SCSS-брекпоинты можно использовать в MediaQueries, но CSS-брекпоинты можно получить JS-ом.

=========================================================

Переменные, которые начинаются с _
должны использоваться внутри фреймворка, но не должны использоваться снаружи него.
